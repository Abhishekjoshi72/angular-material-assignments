Introduction
  1. Angular Material is a User Interface(UI) component library for Angular developers.
  2. Angular Material implements Google's Material Design specifications, providing over 30 UI components that help you build attractive and functional web pages and web applications
  3. Key Features:
    1. Pre-built Theme
    2. UI component library
    3. Compatibility
    4. Accessibility
    5. Internationalisation
    6. Responsive layout
    7. Performance
    8. Integration with Angular
